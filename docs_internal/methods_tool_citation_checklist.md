# Methods Tool Citation Checklist

This checklist lists the tool and source citations needed by `materials_methods_piter_draft.tex`, in the order they appear in the Methods workflow.

## Recommended citations

1. **SRA Toolkit (`prefetch`, `fasterq-dump`)**
   - National Center for Biotechnology Information. *SRA Toolkit*. GitHub. https://github.com/ncbi/sra-tools
   - Supporting documentation: NCBI SRA download guide. https://www.ncbi.nlm.nih.gov/sra/docs/sradownload
   - Note: if the paper prefers database/resource citations over software-repo citations, keep the study accession citation and cite the NCBI SRA documentation for the toolkit commands.

2. **FastQC**
   - Andrews, S. (2010). *FastQC: A quality control tool for high throughput sequence data*. Babraham Bioinformatics. https://www.bioinformatics.babraham.ac.uk/projects/fastqc/

3. **MultiQC**
   - Ewels, P., Magnusson, M., Lundin, S., & Käller, M. (2016). MultiQC: summarize analysis results for multiple tools and samples in a single report. *Bioinformatics, 32*(19), 3047-3048. https://doi.org/10.1093/bioinformatics/btw354

4. **fastp**
   - Chen, S., Zhou, Y., Chen, Y., & Gu, J. (2018). fastp: an ultra-fast all-in-one FASTQ preprocessor. *Bioinformatics, 34*(17), i884-i890. https://doi.org/10.1093/bioinformatics/bty560

5. **STAR**
   - Dobin, A., Davis, C. A., Schlesinger, F., Drenkow, J., Zaleski, C., Jha, S., Batut, P., Chaisson, M., & Gingeras, T. R. (2013). STAR: ultrafast universal RNA-seq aligner. *Bioinformatics, 29*(1), 15-21. https://doi.org/10.1093/bioinformatics/bts635

6. **DESeq2**
   - Love, M. I., Huber, W., & Anders, S. (2014). Moderated estimation of fold change and dispersion for RNA-seq data with DESeq2. *Genome Biology, 15*, 550. https://doi.org/10.1186/s13059-014-0550-8

7. **g:Profiler**
   - Kolberg, L., Raudvere, U., Kuzmin, I., Vilo, J., & Peterson, H. (2023). g:Profiler-interoperable web service for functional enrichment analysis and gene identifier mapping (2023 update). *Nucleic Acids Research, 51*(W1), W207-W212. https://doi.org/10.1093/nar/gkad347
   - Earlier core citation if needed: Raudvere, U., Kolberg, L., Kuzmin, I., Arak, T., Adler, P., Peterson, H., & Vilo, J. (2019). g:Profiler: a web server for functional enrichment analysis and conversions of gene lists (2019 update). *Nucleic Acids Research, 47*(W1), W191-W198. https://doi.org/10.1093/nar/gkz369

8. **Ensembl reference source**
   - Ruffier, M., Kähäri, A., Komorowska, M., et al. (2025). Ensembl 2025. *Nucleic Acids Research, 53*(D1), D948-D957. https://doi.org/10.1093/nar/gkae1071
